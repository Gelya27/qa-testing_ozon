# Тест-кейсы

## Авторизация

1. Вход с валидными данными
   Ожидаемый результат: успешный вход
2. Вход с невалидными данными
   Ожидаемый результат: сообщение об ошибке
3. Вход с пустыми полями
   Ожидаемый результат: сообщение об ошибке

## Поиск и фильтры

1. Поиск товара через строку поиска
   Ожидаемый результат: отображаются релевантные данные
2. Применение фильтра по бренду
   Ожидаемый результат: список товаров соответствует фильтру

## Корзина

1. Добавление товара в корзину
   Ожидаемый результат: товар появляется в корзине
2. Изменение количества товара
   Ожидаемый результат: итоговая сумма перечитывается
3. Удаление товара из корзины
   Ожидаемый результат: товар изчезает из корзины

## Оформление заказа

1. Заполнение формы доставки
   Ожидаемый результат: все обязательные поля проверяются
2. Завершение формления доставки без обязательных полей
   Ожидаемый результат: система не позволяет оформить заказ
