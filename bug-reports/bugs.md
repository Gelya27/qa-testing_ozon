# Баг-репорты

## Баг 1: Счётчик корзины не обновляется после удаления товара

Шаги:
1. Добавить товар в корзину
2. Перейти в корзину  
3. Удалить товар

Ожидаемый результат: счётчик корзины - 0 товаров  
Фактический результат: счётчик показывает 1 товар

Серьёзность: средняя

Приоритет: средний

## Баг 2: Поиск по пустому полю не выдаёт сообщение

Шаги:
1. Нажать поиск с пустой строкой

Ожидаемый результат: сообщение "Введите текст для поиска"  
Фактический результат: страница перезагружается без подсказки

Серьёзность: низкая

Приоритет: средний

## Баг 3: При изменении количества товара в корзине итоговая сумма не всегда обновляется

Шаги:
1. Добавить товар в корзину
2. Увеличить количество на 2

Ожидаемый результат: сумма пересчитана  
Фактический результат: сумма прежняя

Серьёзность: средняя

Приоритет: средний
